# manual

1. 문제마다 하나의 디렉터리를 만듭니다. 디렉터리 안에서 아래 과정을 진행합니다.
2. 문제를 채점하기 전, `풀이과정.md`을 작성합니다.
3. 채점 테스트 케이스를 전부 통과했다면, 다음 문제를 풉니다.
4. 실패했다면 `오답분석.md`를 작성합니다.
5. 문제에 대해 다른 동료에게 피드백을 받고 싶다면, pull request를 열어 피드백을 받습니다.
